# Achar Funções.

Código para encontrar raízes de uma função de forma numérica. A plotagem do gráfico inclui a função e os pontos encontrados como raízes; entretanto, os valores obtidos são retornados ao executar o código no terminal.

Por se tratar de um método numérico, é necessário definir um intervalo, e esse código consegue encontrar raízes de funções que não possuem solução analítica.

Nesse código, foram utilizadas as bibliotecas NumPy, SymPy e Matplotlib.
