## Função Trigonomêtrica

Está disponibilizado tanto o código generalizado para o gráfico de sin(x) e cos(x), limitado a certo intervalo, quanto um pequeno GIF representando o comportamento de ambas ao longo de x (caso se associe x ao tempo). O código da animação também está disponível.

**Código da Animação** → *"Código(Animação Sen, Cos).py"*

**Código do Gráfico** → *"Código(Sen(x),Cos(x)).py"*

**Animação** → *"Seno e Cosseno (Animação).gif"*

**Exemplos de plotagem** → *"seno_cosseno.png"*
